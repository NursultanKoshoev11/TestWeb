# NovaCode Academy — IT Course Landing Page

Одностраничный сайт для IT-курса.

## Что внутри

- `index.html` — структура лендинга.
- `styles.css` — premium dark/cosmic UI, адаптив, glass cards, кнопки, форма, FAQ.
- `script.js` — reveal-анимации и демо-обработка формы.

## Как открыть

Открой файл:

```text
it-course-landing/index.html
```

Можно сразу открыть в браузере без сборки и зависимостей.

## Что заменить перед публикацией

- Название курса.
- Реальные даты старта.
- Реальные цены.
- Контакты.
- Программу курса.
- Отправку формы: backend, Telegram bot, email-сервис или CRM.
